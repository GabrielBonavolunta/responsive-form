📄 Formulário Responsivo

Este projeto consiste em um formulário responsivo desenvolvido utilizando apenas HTML e CSS, com foco em layout moderno, usabilidade e adaptação a diferentes tamanhos de tela.

🚀 Demonstração

O formulário se ajusta automaticamente para dispositivos como:

📱 Smartphones
📲 Tablets
💻 Desktops
🛠️ Tecnologias utilizadas

HTML5

CSS3 (Flexbox)

🎯 Funcionalidades

Layout totalmente responsivo
Campos de entrada estilizados
Botão de envio com feedback visual
Organização clara e intuitiva
Compatível com diferentes navegadores
📱 Responsividade

O projeto utiliza técnicas como:

Media Queries
Flexbox
Unidades relativas (%, rem, vh, vw)

para garantir uma boa experiência em qualquer dispositivo.
